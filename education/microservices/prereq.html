Cloud Native Microservices Pre Requisites

Before you Start

This lab is intended to assist installation of software required for a Java Microservices Lab course. Please skip this lab if you already have the software.!

Package Manager

  1. MacOS:

    Home Brew

  2. Windows:

    We don't have any preferen=nce. Can use Chocolatey

Bash Terminal

  1. MacOS:

    Prefer using iTerm2 iTerm2

  2. Windows:

    We will be using Git Bash>

Git

  1. MacOS:

    Assuming home brew package manager is already installed 

    										brew install git
  2. Windows:

    Git Bash is already installed in previous step. Git Bash is inclusive and gets installed when Git is installed.

Java

  1. MacOS:

    Assuming home brew package manager is already installed 

    										brew cask install java
  2. Windows:

    Download and install Java version 1.8 and above. Prefered to install most recent version.

  3. Verifying version.

    Verify if version installed is used. If not set JAVA_HOME 

    												java -version

IntelliJ

IntelliJ offers a free Community Edition that is sufficient for the course. If you have eclipse or STS, that will work as well. IntelliJ is prefered as it offers support over other editors for building distributed systems.

Docker

  1. MacOS:

    Follow instructions on Docker Getting Started guide to install Docker.
    Either install the Docker toolkit or individual components depending on configuration your machine supports. Note that you need to install VirtualBox if older machines are used. This is described in the getting started guide.
    Getting started guide for Mac is here here
    Once your installation is complete verify installation

  2. Windows:

    Follow instructions on Docker Getting Started guide to install Docker.
    Either install the Docker toolkit or individual components depending on configuration your machine supports. Note that you need to install VirtualBox if older machines are used. This is described in the getting started guide.
    Getting started guide for Windows is here
    Once your installation is complete verify installation

  3. Verify your installation

    
												docker-machine --version
		
												docker --version
													
												docker-machine start
												
												docker-machine env
		
												eval $(docker-machine env)
		
												docker images

Cloud foundry CLI

  1. MacOS:

    Install the Cloud Foundry CLI 

    											brew install cloudfoundry/tap/cf-cli
  2. Windows:

    Navigate to the CF CLI download page and install the latest release. Use the CF CLI commands with Windows Command prompt (not GIT Bash)

Installing PCFDEV

Use the geting started guide for PCFDEV to install PCFDEV.

PCFDEV allows you to develop locally in your machine instead of Public cloud without connectiity.
For most part of this course, PCFDev is sufficient to complete lab exercises.
